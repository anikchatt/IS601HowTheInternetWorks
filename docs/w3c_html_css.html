W3C and HTML and CSS

W3C is short for the world wide web consortium, which is defined as an international community where member organizations, staff and the public work together to develop standards to follow for the web. The mission of the W3C is to lead the world wide web to its full potential by developing protocols and guidelines that will ensure the growth of the web. W3C enables developers to build superior interactive web applications that are available on all devices. For more information on W3C and their mission and standards please visit www.w3.org.

HTML stands for hypertext markup language, and is defined as the standard markup language for creating web pages. Any web page that you see on the Internet is written using some version of HTML code. Utilizing HTML code helps ensure that formatting is standard when using an internet browser. If HTML did not exist there would be no way a browser would be able to decipher how to present text and images. HTML is important because it gives users the ability to publish documents online, retrieve information via hypertext links, and design forms for conducting transactions.

CSS stands for cascading style sheets and is the core technology for building a webpage, along with HTML. CSS is important because it describes how HTML elements are supposed to be displayed. CSS allows users to save time because it can control the layout for multiple web pages simultaneously. The importance of CSS is that it solved the problem of HTML not allowing to contain tags for formatting a web page. The problem with HTML was that when users wanted to add fonts and color attributes, it was a nightmare for web developers, this problem was solved by the W3C because they created CSS which removed the style formatting from the HTML page. CSS saves individuals lots of work because the files are saved as external .css files, which allows the user to change the entire website by changing just one file.
