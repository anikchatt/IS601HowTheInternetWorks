DNS

DNS is short for domain name system, and can be defined as a directory service that is used for transforming alphanumeric domain names into numeric IP addresses. If you would like to learn more about IP addresses and what they are, please navigate to the sidebar presented to the right and read about them there! A simplified way to think of this is that a DNS acts as the phonebook of the internet. Individuals access information online through various domain names, like google.com or espn.com. Web browsers interact through internet protocol addresses, and the DNS translates the domain names to IP addresses so the browser can easily load the Internet page. A basic example of a DNS being utilized in the world would be the domain www.amazon.com which is translated through a DNS into a machine readable IP address 192.0.2.44. DNS servers are crucial because computers cannot process domain names. For a computer to be able to access websites they need to find the relevant IP address connected to the domain name.
